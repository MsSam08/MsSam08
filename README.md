## Core Focus

I build end-to-end machine learning systems supported by scalable data pipelines and cloud infrastructure, draw insight from data to inform real decisions, and teach others to do the same.

My work spans model development to deployment, with a focus on production-ready AI applications. I specialise in combining machine learning, data engineering, and cloud platforms to deliver reliable, automated ML workflows. I build the MLOps scaffolding that keeps AI systems reliable, observable, and continuously improving in production.

Beyond engineering, I teach Data Science and Machine Learning, helping the next generation of practitioners build with confidence.

What makes my work different: **I apply AI where it matters most.** I'm actively building intelligent systems for environmental monitoring, humanitarian impact, and career equity because the world's hardest problems deserve the best infrastructure.

> Designing and delivering high-performance data and AI systems, from insight to action.
